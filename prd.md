# AI驱动的SVG生成器产品需求文档

## 1. 项目概述

### 1.1 产品简介
本项目是一个基于人工智能的SVG矢量图形生成和编辑平台，用户可以通过自然语言描述生成专业的SVG图形，并使用内置的编辑器进行进一步的定制和优化。

### 1.2 产品愿景
成为设计师、开发者和内容创作者首选的AI驱动SVG创作平台，让矢量图形创作变得简单、高效且富有创意。

### 1.3 核心价值
- **AI驱动创作**：通过自然语言描述快速生成专业SVG图形
- **可视化编辑**：提供直观的图形编辑界面和代码编辑器
- **社区共享**：构建创作者社区，促进作品分享和灵感交流
- **多格式导出**：支持SVG、PNG、JPG等多种格式导出

## 2. 目标用户群体

### 2.1 主要用户
- **UI/UX设计师**：需要快速创建图标、插画和界面元素
- **前端开发者**：需要为项目创建自定义图标和矢量图形
- **营销人员**：需要为营销活动创建视觉素材
- **内容创作者**：需要为博客、视频等内容创建配图

### 2.2 用户画像
- **专业设计师李明**：需要高效创建多样化的图标和插画，要求工具功能强大且输出质量高
- **初创公司开发者张伟**：需要快速为产品创建图标，要求操作简单、成本可控
- **自媒体运营者王丽**：需要创建吸引眼球的视觉内容，要求模板丰富、易于使用

## 3. 技术架构

### 3.1 前端技术栈
- **框架**：Next.js 15.0.3 (React 18.3.1)
- **样式**：Tailwind CSS + Radix UI组件库
- **状态管理**：Zustand
- **动画**：Framer Motion
- **图形编辑**：Fabric.js + SVG.js
- **代码编辑器**：Monaco Editor

### 3.2 后端技术栈
- **API**：Next.js API Routes
- **数据库**：PostgreSQL + Prisma ORM
- **认证**：NextAuth.js
- **AI集成**：OpenAI GPT-4、DeepSeek等多模型支持

### 3.3 部署与运维
- **部署平台**：Vercel
- **数据库托管**：PostgreSQL云服务
- **文件存储**：Cloudinary

## 5. 数据库设计

### 5.1 核心数据表

#### 5.1.1 用户表 (users)
```sql
- id: 用户唯一标识
- email: 邮箱地址
- name: 用户名
- password_hash: 密码哈希
- avatar_url: 头像URL
- status: 用户状态 (active/inactive)
- role: 用户角色 (USER/ADMIN)
- created_at: 创建时间
- updated_at: 更新时间
```

#### 5.1.2 SVG作品表 (svgs)
```sql
- id: 作品唯一标识
- user_id: 创作者ID
- title: 作品标题
- description: 作品描述
- svg_content: SVG内容
- prompt_id: 关联的提示词ID
- is_public: 是否公开
- is_official: 是否官方作品
- view_count: 浏览次数
- like_count: 点赞数
- share_count: 分享次数
- favorite_count: 收藏次数
- forward_count: 转发次数
- created_at: 创建时间
- updated_at: 更新时间
```

#### 5.1.3 提示词历史表 (prompt_history)
```sql
- id: 记录唯一标识
- user_id: 用户ID
- content: 提示词内容
- svg_id: 生成的SVG ID
- status: 生成状态 (success/failed)
- error_message: 错误信息
- created_at: 创建时间
```

#### 5.1.4 社区互动表
- **svg_likes**: 点赞记录
- **svg_comments**: 评论记录
- **svg_favorites**: 收藏记录
- **svg_shares**: 分享记录
- **svg_forwards**: 转发记录

### 5.2 关系设计
- 用户与SVG作品：一对多关系
- SVG作品与互动记录：一对多关系
- 用户与互动记录：一对多关系
- 提示词与SVG作品：一对多关系

## 6. API接口设计

### 6.1 认证相关API
- `POST /api/auth/signin` - 用户登录
- `POST /api/auth/signup` - 用户注册
- `POST /api/auth/signout` - 用户登出

### 6.2 SVG相关API
- `GET /api/my-svgs` - 获取用户SVG列表
- `POST /api/svgs` - 创建新SVG
- `PUT /api/svgs/[id]` - 更新SVG
- `DELETE /api/svgs/[id]` - 删除SVG
- `POST /api/svgs/[id]/like` - 点赞SVG

### 6.3 社区相关API
- `GET /api/community` - 获取社区SVG列表
- `GET /api/templates` - 获取模板列表
- `POST /api/community/[id]/comment` - 添加评论

### 6.4 AI生成API
- `POST /api/generate` - AI生成SVG
- `POST /api/chat` - AI对话接口
